    古猿诗词

    其一

    古猿东望路漫漫，初一七班香蕉烂。
    古猿不交实验班，海哥变成白眼怪。
    古猿回归博物馆，海哥流放小和山。
    小和山上香蕉怪，直骂古猿觉悟慢。

    其二

    饮（小米）酒 其五

    结庐在杭外，常听汤姆喧。
    问君何能尔？猿远地自偏。
    采蕉七班下，悠然见和山。
    山气日夕佳，古猿相与还。
    此中有觉悟，欲辨已忘言。

      ——小土（王昊）宸

    其三

    古猿值日分数危，初一七班全是灰。
    若待下午海哥归，古猿飞进垃圾堆。

    其四

    古猿上课不带书，文综考试把毛梳。
    一题不会就翻书，作弊零分只会哭。

    其五

    百猿进门就猴叫，全班同学都大笑。
    海哥看见不太妙，建议全班都通校。

    其六

    海哥让答问题，结果猿打喷嚏。
    引起海哥注意，但是古猿不立。

    其七

    古猿上课傻笑，海哥建议通校。
    古猿知不太妙，回到家吃香蕉。

    其八

    古猿冰上爬，压强是几帕？
    海哥把他骂，只能找小马。

    其九

    古猿夫如何，放屁停不了。
    臭屁钟古猿，屎尿割昏晓。
    肠道生臭屁，熏死群归鸟。
    会当珠峰顶，一屁众生倒。

    其十

    古猿画图不用尺，上课乱叫尽作死。
    花心古猿太无耻，吃完香蕉想吃屎。

    其十一

    古猿喜欢王圣茹，天天大叫：“土番薯！”
    她说古猿太粗鲁，古猿大叫：“肏你母！”
